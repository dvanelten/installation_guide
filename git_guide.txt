- Repository in GitHub erstellen
- lokal beliebigen Ordner aussuchen, wo Repository liegen soll
(Dwsktop/master-thesis/git_repository)
$ git clone <http://github.com/$USER/$NAME.git>

- Daten zum Repository hinzufuegen:
$ git add <name>
$ git commit -m <comment>
$ git push

- Status überprüfen: 
$ git status

- txt.Datei erstellen
$ touch <name>

- Datei im Repository + lokal loeschen:
$ git rm <name>
